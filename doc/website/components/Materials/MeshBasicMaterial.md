# MeshBasicMaterial
一个以简单着色（平面或线框）方式来绘制几何体的材质。
这种材质不受光照的影响。

| 属性名        | 是否必须           | 默认值  |含义  |
| ------------- |:-------------:| -----:| -----:|
| color          | No | 0xffffff | 材质颜色 |
| alphaMap      | No      |   null  |透明度贴图|
| meshName | No      |   undefind |在 FBXLoader 组件替换指定名称模型的材质，FBX文件包含多个模型 |
